# power-analytics.appspot.com

## Develop
### Environment
- Node.js
- Python2.7, Bottle Framework
- Google App Engine

### Install
```
$ git clone git@github.com:1987yama3/power-analytics.appspot.com.git
$ npm install -g gulp
$ npm install
```

### Build
```
$ gulp build
```

### Deploy
```
$ gulp deploy
```
